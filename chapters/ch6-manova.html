多變量變異數分析 (MANOVA)

Ch.6 同時比較多個依變數的組間差異

一、從 ANOVA 到 MANOVA

ANOVA（變異數分析）是檢驗一個依變數（metric）在不同組別（nonmetric 自變數）之間是否有顯著差異的方法。但當研究涉及多個依變數時，需要使用 MANOVA。

為什麼不對每個依變數分別做 ANOVA？

  • 型一錯誤膨脹：對 p 個依變數分別做 ANOVA，實際犯錯機率遠大於 α = .05
  • 忽略依變數間的相關：分別分析會遺失依變數之間的共變關係資訊
  • 多變量效果：MANOVA 可以偵測到個別 ANOVA 無法發現的組間差異模式

ANOVA tests whether one dependent variable (metric) differs significantly across groups (nonmetric IVs). When research involves multiple DVs, MANOVA is needed.

Why not run separate ANOVAs for each DV?

  • Inflated Type I error: Running p separate ANOVAs raises the actual error rate well above α = .05
  • Ignores DV correlations: Separate analyses lose information about covariance between DVs
  • Multivariate effects: MANOVA can detect group difference patterns that individual ANOVAs miss

白話解釋

ANOVA 就像看一個成績（例如數學）在不同班級之間是否有差異。MANOVA 則是同時看多個成績（數學、英文、自然）在不同班級之間是否有差異，而且還考慮了這些成績之間的關聯性。就像老師不只看單科成績，而是綜合評估學生的整體表現。

ANOVA is like checking whether one grade (e.g., math) differs across classes. MANOVA simultaneously checks multiple grades (math, English, science) across classes while considering how these grades relate to each other. It is like a teacher evaluating overall student performance rather than just one subject.

二、MANOVA 的統計原理

MANOVA 的核心邏輯與 ANOVA 相同：比較組間變異與組內變異的比率。差異在於 MANOVA 處理的是矩陣而非單一數值。

MANOVA's core logic mirrors ANOVA: comparing the ratio of between-group variance to within-group variance. The difference is that MANOVA works with matrices rather than single values.

虛無假設

ANOVA 的 H₀：μ₁ = μ₂ = ... = μg（各組在單一依變數上的平均數相等）

MANOVA 的 H₀：各組的平均數向量（mean vectors）相等，即所有依變數的聯合分布在各組之間沒有差異。

ANOVA H₀: μ₁ = μ₂ = ... = μg (group means on a single DV are equal)

MANOVA H₀: The mean vectors are equal across groups — the joint distribution of all DVs does not differ between groups.

組間 vs. 組內變異矩陣

  • 組間變異-共變數矩陣（B）：反映各組平均值的差異程度（類似 ANOVA 的 SSB）
  • 組內變異-共變數矩陣（W）：反映各組內部的變異程度（類似 ANOVA 的 SSW）
  • MANOVA 檢定統計量基於 B 與 W 的某種比率函數
  • Between-group variance-covariance matrix (B): Reflects group mean differences (analogous to ANOVA's SSB)
  • Within-group variance-covariance matrix (W): Reflects within-group variability (analogous to ANOVA's SSW)
  • MANOVA test statistics are based on some function of the B to W ratio

三、多變量檢定統計量

MANOVA 提供四種常用的檢定統計量，各有不同的特性與適用情境。所有統計量都可以轉換為近似 F 值來做顯著性檢定。

MANOVA offers four common test statistics, each with different properties and contexts. All can be converted to approximate F-values for significance testing.

統計量 計算原理 穩健性 適用時機
Wilks' Lambda (Λ) 未被解釋的變異比例 中等；最常用 一般情況首選
Pillai's Trace 已解釋變異的加總 最穩健 假設違反時
Hotelling's Trace 特徵值的加總 對假設敏感 兩組比較
Roy's Largest Root 僅最大特徵值 最不穩健 差異集中時

Wilks' Lambda 公式

Λ = |W| / |B + W| |W| = 組內矩陣行列式, |B+W| = 總變異矩陣行列式

Λ 的範圍為 0 到 1。Λ = 1 表示完全沒有組間差異，Λ 越接近 0 表示組間差異越大。

Λ ranges from 0 to 1. Λ = 1 means no group differences; Λ closer to 0 indicates larger group differences.

如何選擇統計量？

  • 一般報告首選：Wilks' Lambda（最常用、期刊接受度最高）
  • 樣本小或假設有疑慮：Pillai's Trace（最穩健）
  • 四種統計量結論一致：通常可安心報告任一種
  • 四種統計量結論不一致：深入檢查假設是否被違反，傾向採用 Pillai's Trace
  • Default reporting: Wilks' Lambda (most common, widely accepted in journals)
  • Small samples or assumption concerns: Pillai's Trace (most robust)
  • All four statistics agree: safe to report any one
  • Statistics disagree: investigate assumption violations; prefer Pillai's Trace

四、假設檢驗流程

MANOVA 的假設檢驗遵循一個從整體到個別的邏輯順序。必須先通過整體檢定，才能進一步探查個別依變數的差異。

MANOVA hypothesis testing follows a general-to-specific logical sequence. The omnibus test must be significant before probing individual DV differences.

圖：MANOVA 假設檢驗流程 — 從整體到個別的階層式分析

各步驟詳細說明

  1. 假設檢查：多變量常態性、Box's M 檢定（共變數矩陣同質性）、充足的樣本大小
  2. 整體 MANOVA 檢定：若 Wilks' Λ 顯著（p < .05），表示至少有一個依變數在組間有差異
  3. 個別 ANOVA（Univariate F-tests）：對每個依變數分別進行 F 檢定，找出哪些依變數有顯著差異。建議使用 Bonferroni 修正（α/p）控制型一錯誤
  4. 事後比較（Post-hoc）：對顯著的依變數進行成對組別比較（如 Tukey HSD、Bonferroni），找出具體差異所在
  5. 效果量計算：使用 partial η² 衡量效果的實質大小
  1. Assumption checks: Multivariate normality, Box's M test (homogeneity of covariance matrices), adequate sample size
  2. Omnibus MANOVA test: If Wilks' Λ is significant (p < .05), at least one DV differs across groups
  3. Univariate F-tests: Separate F-test for each DV to identify which have significant group differences. Use Bonferroni correction (α/p) to control Type I error
  4. Post-hoc comparisons: Pairwise group comparisons (e.g., Tukey HSD, Bonferroni) on significant DVs to locate specific differences
  5. Effect size: Use partial η² to assess practical significance

五、效果量

統計顯著性（p 值）只告訴我們效果是否「存在」，但不告訴我們效果有多「大」。效果量（effect size）提供了實質意義的衡量標準。

Statistical significance (p-value) only tells us whether an effect "exists," not how "large" it is. Effect size provides a measure of practical significance.

Partial η²（偏 Eta 平方）

partial η² = SSeffect / (SSeffect + SSerror) 該效果解釋的變異占（效果 + 誤差）的比例
Partial η² 效果大小 解釋
.01 小效果 約 1% 變異
.06 中效果 約 6% 變異
.14 大效果 約 14%+ 變異

效果量的重要性

  • 大樣本中，即使微不足道的差異也可能統計顯著（p < .05），但效果量可能極小（η² < .01）
  • APA 規範要求報告效果量，不能僅報告 p 值
  • partial η² 與 η² 的差異：partial η² 控制了其他效果的影響，在多因子設計中更為適切
  • With large samples, even trivial differences can be statistically significant (p < .05), while effect size may be negligible (η² < .01)
  • APA guidelines require reporting effect sizes, not just p-values
  • Partial η² vs. η²: partial η² controls for other effects, more appropriate in multi-factor designs

六、MANOVA 的假設

MANOVA 的假設比 ANOVA 更為嚴格，因為涉及多個依變數。違反假設可能導致結果不可靠。

MANOVA assumptions are stricter than ANOVA's because of the multiple DVs involved. Violations may lead to unreliable results.

1. 多變量常態性

依變數的聯合分布應呈多變量常態分布（不僅僅是每個依變數個別常態）。

檢驗方式：Mardia's test、多變量 Q-Q plot。

穩健性：大樣本（每組 ≥ 20-30）時對輕微違反具有穩健性。偏態比峰度的影響更大。

The joint distribution of DVs should be multivariate normal (not just individually normal).

Test: Mardia's test, multivariate Q-Q plot.

Robustness: Robust to mild violations with large samples (≥ 20-30 per group). Skewness has more impact than kurtosis.

2. 共變數矩陣同質性

各組的變異數-共變數矩陣應該相等（類似 ANOVA 的變異數同質性假設的多變量版本）。

檢驗方式：Box's M 檢定。

注意：Box's M 對常態性非常敏感，大樣本時容易顯著。一般建議若 p < .001 才視為嚴重違反。

違反時的對策：使用 Pillai's Trace（最穩健）；確保各組樣本大小相等。

The variance-covariance matrices should be equal across groups (multivariate extension of ANOVA's homogeneity of variance).

Test: Box's M test.

Note: Box's M is very sensitive to normality and easily significant with large samples. Generally treat p < .001 as a serious violation.

If violated: Use Pillai's Trace (most robust); ensure equal group sizes.

3. 樣本大小

充足的樣本大小是 MANOVA 統計檢力的關鍵。

基本要求：

  • 每組的觀察值數目必須大於依變數的數目
  • 建議：每個 cell 至少 20 個觀察值
  • 各組樣本大小盡量相等（平衡設計更穩健）

Adequate sample size is critical for MANOVA's statistical power.

Requirements:

  • Number of observations per group must exceed the number of DVs
  • Recommended: at least 20 observations per cell
  • Equal group sizes preferred (balanced designs are more robust)

4. 依變數之間的相關

依變數之間應有適度的相關（通常 .2 到 .6），MANOVA 才有意義。

太低（< .2）：使用 MANOVA 沒有優勢，不如分別做 ANOVA

太高（> .9）：依變數幾乎是同一個概念，具有多元共線性問題

DVs should be moderately correlated (typically .2 to .6) for MANOVA to be meaningful.

Too low (< .2): No advantage over separate ANOVAs

Too high (> .9): DVs are nearly the same concept; multicollinearity issues

Box's M 檢定的解讀指南

  • Box's M 不顯著（p > .001）：假設成立，繼續進行 MANOVA
  • Box's M 顯著但各組樣本大小相等：對結果影響較小，可使用 Pillai's Trace
  • Box's M 顯著且各組樣本不等：需謹慎解釋結果；考慮使用非參數替代方法
  • Box's M not significant (p > .001): Assumption met, proceed with MANOVA
  • Box's M significant but equal group sizes: Minimal impact; use Pillai's Trace
  • Box's M significant with unequal groups: Interpret cautiously; consider nonparametric alternatives

方法比較

ANOVA（變異數分析）

  • 依變數：1 個（metric）
  • 自變數：1 個或多個（nonmetric / categorical）
  • 檢定統計量：F-test
  • 假設：常態性、變異數同質性（Levene's test）
  • 適用時機：比較各組在單一結果變數上的差異

ANOVA (Analysis of Variance)

  • DV: 1 (metric)
  • IV: 1 or more (nonmetric / categorical)
  • Test statistic: F-test
  • Assumptions: Normality, homogeneity of variance (Levene's test)
  • When to use: Comparing groups on a single outcome variable

MANOVA（多變量變異數分析）

  • 依變數：2 個或以上（metric）
  • 自變數：1 個或多個（nonmetric / categorical）
  • 檢定統計量：Wilks' Λ, Pillai's Trace, Hotelling's Trace, Roy's Largest Root
  • 假設：多變量常態性、共變數矩陣同質性（Box's M）
  • 適用時機：同時比較各組在多個相關結果變數上的差異

MANOVA (Multivariate Analysis of Variance)

  • DVs: 2 or more (metric)
  • IV: 1 or more (nonmetric / categorical)
  • Test statistics: Wilks' Λ, Pillai's Trace, Hotelling's Trace, Roy's Largest Root
  • Assumptions: Multivariate normality, homogeneity of covariance matrices (Box's M)
  • When to use: Simultaneously comparing groups on multiple related outcome variables

重複量測 ANOVA / MANOVA

  • 特點：同一受試者在不同時間點或條件下被重複測量
  • 額外假設：球形性假設（Sphericity，Mauchly's test）— 適用於重複量測 ANOVA
  • 替代方案：若球形性被違反，可使用多變量方法（將時間點視為多個依變數）或 Greenhouse-Geisser / Huynh-Feldt 修正
  • 適用時機：前測-後測設計、縱貫性研究、受試者內設計

Repeated Measures ANOVA / MANOVA

  • Feature: Same subjects measured repeatedly across time points or conditions
  • Additional assumption: Sphericity (Mauchly's test) — applies to RM-ANOVA
  • Alternative: If sphericity is violated, use multivariate approach (time points as multiple DVs) or Greenhouse-Geisser / Huynh-Feldt corrections
  • When to use: Pre-post designs, longitudinal studies, within-subjects designs

關鍵名詞解釋

平均數向量（Mean Vector / Centroid）

一組的所有依變數平均值所組成的向量。在 ANOVA 中我們比較各組的平均數（一個數值），在 MANOVA 中我們比較的是各組的平均數向量（一組數值）。MANOVA 的虛無假設就是所有組的平均數向量（centroid）相等。

A vector composed of a group's means on all DVs. In ANOVA we compare group means (a single number); in MANOVA we compare group mean vectors (a set of numbers). MANOVA's null hypothesis is that all group centroids are equal.

型一錯誤膨脹（Inflated Type I Error）

當對多個依變數分別進行假設檢定時，即使每次檢定的顯著水準為 α = .05，整體犯型一錯誤的機率會遠大於 .05。例如：對 5 個依變數分別做 ANOVA，整體型一錯誤率約為 1-(1-.05)⁵ ≈ .23，即有 23% 的機率至少犯一次型一錯誤。MANOVA 透過單一多變量檢定避免了此問題。

When conducting separate tests on multiple DVs, even if each test uses α = .05, the overall Type I error probability far exceeds .05. For example: 5 separate ANOVAs yield an overall rate of about 1-(1-.05)⁵ ≈ .23, a 23% chance of at least one Type I error. MANOVA avoids this by using a single multivariate test.

Wilks' Lambda (Λ)

MANOVA 最常用的檢定統計量。Λ 衡量的是「組間差異未能解釋的變異比例」。Λ = 1 表示各組完全沒有差異，Λ 越小（接近 0）表示組間差異越大。Λ 可以轉換為近似 F 值進行顯著性檢定。在穩健性方面處於中等水準。

MANOVA's most commonly used test statistic. Λ measures the proportion of variance not explained by group differences. Λ = 1 means no group differences; smaller Λ (approaching 0) indicates larger group differences. Λ can be converted to an approximate F-value for significance testing. It has moderate robustness.

共變數矩陣同質性（Homogeneity of Covariance Matrices）

MANOVA 的重要假設之一，要求各組的變異數-共變數矩陣相等。這是 ANOVA 中變異數同質性假設（Levene's test）的多變量推廣。透過 Box's M 檢定來評估。由於 Box's M 對非常態性非常敏感，通常以 p < .001 作為判斷門檻。

A key MANOVA assumption requiring that variance-covariance matrices are equal across groups. This is the multivariate extension of ANOVA's homogeneity of variance assumption (Levene's test). Assessed via Box's M test. Since Box's M is very sensitive to non-normality, p < .001 is typically used as the threshold.

特徵值（Eigenvalue）

在 MANOVA 中，特徵值來自 W⁻¹B 矩陣（組內矩陣的逆矩陣乘以組間矩陣）。特徵值代表在各個正交維度上，組間差異相對於組內變異的比率。特徵值越大，表示該維度上的組間差異越明顯。Hotelling's Trace 是所有特徵值的加總，Roy's Largest Root 僅考慮最大特徵值。

In MANOVA, eigenvalues come from the W⁻¹B matrix (inverse of within-group matrix times between-group matrix). They represent the ratio of between-group to within-group variance on each orthogonal dimension. Larger eigenvalues indicate more pronounced group differences on that dimension. Hotelling's Trace sums all eigenvalues; Roy's Largest Root uses only the largest.
