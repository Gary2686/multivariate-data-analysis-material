多元判別分析

Ch.7 預測群體成員隸屬關係的多變量技術

一、判別分析概述

多元判別分析（MDA）是一種依變數為類別變數（非計量）、自變數為計量變數的多變量技術。其核心目的是：

  • 找出最能區辨（discriminate）不同群體的自變數組合
  • 建立判別函數（discriminant function），用來將新觀察值分類到已知群體
  • 了解哪些自變數對分類最具貢獻

Multiple Discriminant Analysis (MDA) is a multivariate technique where the dependent variable is categorical (nonmetric) and the independent variables are metric. Its core purposes are:

  • Identify the combination of IVs that best discriminates among groups
  • Build discriminant functions to classify new observations into known groups
  • Understand which IVs contribute most to classification

白話解釋

判別分析就像海關人員分辨旅客類別。海關人員觀察旅客的護照國籍、行李數量、停留天數等多項特徵（自變數），綜合判斷旅客屬於「觀光客」、「商務旅客」還是「留學生」（類別依變數）。判別分析做的就是找出哪些特徵組合最能準確分辨旅客類別，並建立一套分類規則。

Discriminant analysis is like a customs officer classifying travelers. The officer observes passport nationality, number of bags, length of stay, and other characteristics (IVs), then determines whether a traveler is a "tourist," "business traveler," or "student" (categorical DV). MDA finds the combination of features that best separates these groups and builds a classification rule.

MDA vs. MANOVA

MDA 與 MANOVA 使用相同的數學運算，但目的不同：

  • MANOVA：檢驗群組間的平均數是否有顯著差異（假說檢定取向）
  • MDA：建立分類規則，預測新觀察值屬於哪個群組（預測取向）

MDA and MANOVA use the same mathematics but with different goals:

  • MANOVA: Tests whether group means differ significantly (hypothesis testing)
  • MDA: Builds classification rules to predict group membership (prediction)

二、判別函數

判別函數是自變數的線性組合，其形式類似迴歸方程式，但目的是最大化群體間的差異。

A discriminant function is a linear combination of IVs, similar in form to a regression equation, but designed to maximize between-group differences.

判別函數方程式

D = b1X1 + b2X2 + ... + bkXk D = 判別分數, b = 判別係數（權重）, X = 自變數

重心（Centroid）：每個群體在判別函數上的平均判別分數。兩組判別分析中，若兩組重心距離越遠，判別函數的分類效果越好。

最大函數數量：可以萃取的判別函數數量 = min(群組數 - 1, 自變數數量)。

  • 兩組判別分析：最多 1 個判別函數
  • 三組判別分析、5 個自變數：最多 2 個判別函數

Centroid: The mean discriminant score for each group. In two-group MDA, the farther apart the centroids, the better the function separates the groups.

Maximum number of functions: Number of discriminant functions = min(groups - 1, predictors).

  • Two-group MDA: at most 1 discriminant function
  • Three-group MDA with 5 IVs: at most 2 discriminant functions

圖：兩組判別分析 -- 群體重心與分類邊界

兩組判別分析

最基本的判別分析形式，依變數只有兩個類別（如：購買/不購買、通過/未通過）。

  • 僅產生 1 個判別函數
  • 分類依據：計算判別分數 D，與切割分數（cutting score）比較
  • 若兩組樣本數相等：切割分數 = 兩組重心的中點
  • 若兩組樣本數不等：使用加權切割分數 ZCE = (NAZB + NBZA) / (NA + NB)

Two-Group Discriminant Analysis

The most basic form of MDA, where the DV has only two categories (e.g., purchase/no purchase, pass/fail).

  • Produces only 1 discriminant function
  • Classification: compute discriminant score D, compare with cutting score
  • Equal group sizes: cutting score = midpoint of the two centroids
  • Unequal group sizes: weighted cutting score ZCE = (NAZB + NBZA) / (NA + NB)

多組判別分析

依變數有三個或以上類別（如：高/中/低忠誠度客戶）。

  • 可產生 多個判別函數，最多 min(g-1, p) 個
  • 各判別函數彼此正交（orthogonal），互相獨立
  • 第一個函數解釋最多的群體間差異，第二個解釋次多，以此類推
  • 分類依據：計算每個觀察值到各組重心的距離，歸類到最近的群組
  • 需要使用分類函數（classification functions），為每個群體建立一組係數

Multiple-Group Discriminant Analysis

DV has three or more categories (e.g., high/medium/low loyalty customers).

  • Can produce multiple discriminant functions, up to min(g-1, p)
  • Functions are orthogonal (uncorrelated with each other)
  • First function explains the most between-group variance, second explains the next most, etc.
  • Classification: compute distance from each observation to group centroids, assign to nearest
  • Uses classification functions with a separate set of coefficients for each group

三、統計顯著性

判別函數建立後，必須評估其是否具有統計顯著性。主要使用三個指標：

After deriving discriminant functions, we must assess their statistical significance using three key metrics:

Wilks' Lambda (Λ)

定義：群體內變異數 / 總變異數。衡量群體間重疊程度。

範圍：0 到 1。越接近 0 表示群體間差異越大（判別函數越好）。

檢定：以卡方轉換進行顯著性檢定。

Wilks' Lambda is the ratio of within-group to total variability. Closer to 0 = better discrimination.

Definition: Ratio of within-group to total variability. Measures group overlap.

Range: 0 to 1. Closer to 0 means greater between-group difference (better function).

Test: Converted to chi-square for significance testing.

特徵值 (Eigenvalue)

定義：群體間變異數 / 群體內變異數的比值。

解釋：值越大表示判別函數解釋的群體差異越多。

用途：計算各判別函數解釋的變異量百分比。

Eigenvalue is the ratio of between-group to within-group variance. Larger = better.

Definition: Ratio of between-group to within-group variance.

Interpretation: Larger values indicate the function accounts for more group differences.

Use: Calculate the percentage of variance explained by each function.

典型相關 (Canonical Correlation)

定義：判別分數與群組成員之間的相關程度。

Rc²：判別函數所解釋的依變數變異量百分比（類似 R²）。

計算：Rc² = 特徵值 / (1 + 特徵值)

Canonical correlation measures the association between discriminant scores and group membership.

Definition: Correlation between discriminant scores and group membership.

Rc²: Percentage of DV variance explained by the function (analogous to R²).

Formula: Rc² = Eigenvalue / (1 + Eigenvalue)

係數解釋方法

解釋各自變數對判別函數的貢獻時，可使用三種方法：

  • 標準化判別係數：類似標準化迴歸係數（Beta），考量尺度差異後比較各變數的相對重要性
  • 結構矩陣（Structure Matrix）：各自變數與判別函數的相關（也稱為判別負荷量）。通常被認為較係數更可靠
  • 偏 F 比率：類似逐步迴歸中各變數的 F 值，反映個別區辨能力

Three approaches for interpreting each IV's contribution:

  • Standardized discriminant weights: Like standardized Beta in regression, compare relative importance across variables
  • Structure matrix: Correlations of each IV with the discriminant function (also called discriminant loadings). Generally considered more reliable than weights
  • Partial F-ratios: Like stepwise regression F-values, reflecting individual discriminating power

四、分類準確度

判別分析的最終目的是正確分類，因此必須評估分類的準確度。

The ultimate goal of MDA is correct classification, so we must evaluate classification accuracy.

混淆矩陣 預測群組 合計
實際群組 群組 1 群組 2
群組 1 正確分類 (a) 誤分類 (b) a + b
群組 2 誤分類 (c) 正確分類 (d) c + d

命中率與統計檢定

命中率 (Hit Ratio) = (a + d) / (a + b + c + d) × 100% 正確分類的觀察值佔全體比例

命中率必須顯著高於隨機分類。判斷基準包括：

  • 最大機率準則：命中率應超過最大群組的比例（例如群組 1 佔 60%，命中率應 > 60%）
  • 比例機率準則：命中率應超過各群組比例平方和（p₁² + p₂² + ...）
  • 實務標準：命中率應至少超過隨機準則 25%

The hit ratio must be significantly better than chance. Benchmarks include:

  • Maximum chance criterion: Hit ratio must exceed the largest group's proportion (e.g., if Group 1 = 60%, hit ratio > 60%)
  • Proportional chance criterion: Hit ratio must exceed sum of squared proportions (p₁² + p₂² + ...)
  • Practical standard: Hit ratio should exceed the chance criterion by at least 25%

Press's Q 統計量

Press's Q = [N - (n × k)]² / [N(k - 1)] N = 總樣本數, n = 正確分類數, k = 群組數

Press's Q 值與自由度為 1 的卡方臨界值（3.84）比較。若 Q > 3.84，表示分類結果顯著優於隨機分類。

Compare Press's Q to the chi-square critical value with 1 df (3.84). If Q > 3.84, classification is significantly better than chance.

交叉驗證（Cross-Validation）：為避免過度樂觀的命中率，應採用留一法（leave-one-out）交叉驗證。將每個觀察值依次留出，用剩餘樣本建立判別函數後分類該觀察值。交叉驗證的命中率通常略低於原始命中率，但更能反映真實的分類能力。若兩者差距過大，可能存在過度配適問題。

Cross-Validation: To avoid overly optimistic hit ratios, use leave-one-out cross-validation. Each observation is classified by the function derived from all other cases. The cross-validated hit ratio is typically slightly lower but reflects true classification ability. A large gap between original and cross-validated rates suggests overfitting.

假設 內容說明 違反後果 檢驗方式
多元常態性 自變數在各群體內呈多元常態分配 影響統計檢定的準確性 單變量常態性檢定、散佈圖

Box's M 檢定注意事項

Box's M 檢定對多元常態性的違反非常敏感。在大樣本中，即使共變異數差異很小也可能顯著。因此應搭配樣本數比例來判斷：若各組樣本數大致相等，即使 Box's M 顯著，判別函數仍具穩健性。

Box's M is very sensitive to departures from multivariate normality. In large samples, even trivial differences become significant. Use with group size ratios: if group sizes are roughly equal, the function remains robust even with a significant Box's M.

六、逐步判別分析

當自變數眾多時，可採用逐步判別分析來挑選最重要的自變數。逐步法依序選入對分類最有貢獻的變數，直到沒有變數能顯著改善分類為止。

When there are many IVs, stepwise MDA selects the most important variables. It sequentially enters variables that contribute most to classification until no variable significantly improves it.

選取準則 原理 特點
Wilks' Lambda 選入使 Lambda 最小化的變數 最常用，考慮整體分類效果
Mahalanobis D² 選入使各組間距離最大化的變數 著重群體間的距離最大化
Rao's V 選入使 V 值增加最多的變數 考量群體間整體差異性

八、關鍵名詞解釋

判別函數（Discriminant Function）

自變數的線性組合，用來計算「判別分數」，以最大化群體間的差異。每個函數都是正交的，依序解釋遞減的群體間變異量。

A linear combination of IVs used to compute a "discriminant score" that maximizes between-group differences. Each function is orthogonal and explains decreasing amounts of between-group variance.

重心（Centroid）

各群體在判別函數空間中的「平均位置」，即該群體所有成員的判別分數平均值。兩組分析中，分類的切割點即落在兩個重心之間。

The "average position" of each group in the discriminant function space. In two-group analysis, the classification cutting point falls between the two centroids.

結構矩陣（Structure Matrix）

各自變數與判別函數之間的簡單相關，也稱為「判別負荷量」。相較於標準化判別係數，結構矩陣被認為是解釋各變數區辨貢獻更可靠的方法，因為它不受多元共線性影響。

Simple correlations between each IV and the discriminant function, also called "discriminant loadings." Considered more reliable than standardized weights for interpreting each variable's discriminating contribution, as it is not affected by multicollinearity.

混淆矩陣（Confusion Matrix）

也稱為分類矩陣，以交叉表的形式呈現實際群組與預測群組的對應關係。對角線上的數字為正確分類數，非對角線為誤分類數。藉此計算命中率與各群組的分類正確率。

Also called a classification matrix, it shows the cross-tabulation of actual versus predicted group membership. Diagonal values are correct classifications; off-diagonal values are misclassifications. Used to compute hit ratio and per-group accuracy.

切割分數（Cutting Score）

在兩組判別分析中，用來決定將觀察值分入哪個群組的門檻值。判別分數高於切割分數分入一組，低於則分入另一組。等樣本時為兩個重心的中點；不等樣本時使用加權公式計算。

In two-group MDA, the threshold for assigning observations to groups. Scores above the cutting score go to one group; below to the other. For equal groups, it is the midpoint of centroids; for unequal groups, a weighted formula is used.
